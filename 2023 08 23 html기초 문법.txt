<h1> ~ <h2> : 제목을 입력할 때 사용
<br> : 개행
<hr> : 가로 줄 긋기
&nbsp : 띄어쓰기
<p> : 글 작성, 종료 시 줄 바꿈
<pre> : 글 작성, 문자열 형식 그대로 출력
<b>, <strong> : 글자 굴게 표시, 강조
<i>, <em> : 글자 기울여 표시
<mark> : 형광팬 효과
<u> : 밑줄
<s> : 취소선
<small> : 글자를 작게
<abbr title="~~"> : 마우스 오버 시 툴팁 출력
<ul> : 순서없는 목록을 작성
<ol> : 순서있는 목록을 작성
<li> : 목록에 들어가는 내용을 작성
<style> : HTML 문서의 스타일(크기, 모양, 색상 등)을 지정 == CSS
<table> : 행과 열로 이루어진 표 만들기 --> 클래스명 지정 가능
<tr> : 한 행을 나타냄
<td> : 한 컬럼을 나타냄
<th> : td와 기본은 같지만 추가적으로 굵은 글씨, 가운데 정렬
<caption> : 테이블의 제목이나, 설명 내용을 추가
border : 선 굵기
width : 가로 크기
height : 세로 크기
rowspan : 상하 병합
colspan : 좌우 병합
thead : 테이블의 상단 부분 명 (컬럼명)
tbody : 테이블의 중단 부분 영역 (실제 값, 내용)
tfoot : 테이블의 하단 부분 영역 (합계)
