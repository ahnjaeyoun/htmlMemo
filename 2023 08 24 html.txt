이미지 무료 사이트 : https://unsplash.com/ko
폰트(한글) 무료 사이트 : https://noonnu.cc/

<img> : 이미지 삽입
width / height 속성 : 크기를 지정하는 속성
alt 속성: 이미지를 설명하는 속성 -> 이미지가 출력되지 않는 경우 화면에 직접 표시
src 속성 : 재생할 미디어 파일의 URL을 명시
고정 크기 단위(px) : 화면의 크기와 상관없이 지정된 값의 크기를 갖는 단위
가변 크기 단위(%) : 화면이나 영역 기준 사이트 크기에 따라 크기가 변경되는 단위

<a> : 현재 HTML문서와 다른 HTML문서를 연결해서 페이지 이동을 할 수 있게하는 기능을 제공

<input> : 웹 문서에서 사용자가 입력할 수 있는 양식을 제공하는 태그
size 속성 : 입력상자 크기
maxlength 속성 : 입력 받는 텍스트의 최대길이
placeholder 속성 : 입력 상자에 작성될 내용을 설명
password 타입 : 비밀번호 입력 상자(입력되는 텍스트를 가려줌)
url 타입 : 주소 입력 시 사용하는 입력 상자
tel 타입 : 전화번호 입력 시 사용하는 입력 상자
email 타입 : 이메일 입력 시 사용하는 입력 상자
number 타입 : 숫자만 입력
range 타입 : 슬라이드 바를 이용해서 숫자 지정(단독 사용X, JS와 함께 사용)
날짜/시간 관련 타입들 date, month, week, time, datetime-local
name 속성 : 묶음 표시, input 태그 값의 이름 지정
radio 타입 : 선택 가능한 란 표시(원형)
checkbox 타입 : 선택 가능한 란 표시(박스형)
color 타입 : 색상 지정
file 타입 : 파일 삽입
hidden 타입 : 값을 숨길 때 사용
value 속성 : input태그에 value값 지정

<label> : 결합 또는 결속을 위해 사용
for 속성 : 결합

<form> : block 형식, 내부에 작성된 input 태그의 값을 서버로 전달하는 역할
action 속성 : 내부에 작성된 값을 전달할 서버 또는 페이지 주소를 작성
method(방식) 속성 : 서버로 데이터 전달하는 방식을 지정
name 속성 : form 태그의 이름을 지정하여 form 태그 구분(JS 사용)
target 속성 : action에 지정된 주소를 현재탭/새탭 중 어디서 열지 지정

<button> : 버튼을 나타냄
submit 타입 : 제출
reset 타입 : 초기화

<fieldset> : 테두리를 만들어 그룹을 구분하는 영역 생성
<legend> : fieldset 테두리에 이름을 붙임

<textarea> : 여러줄을 입력하기 위한 입력상자
