<link rel="stylesheet" href="css/selector2.css" type="text/css"> : CSS파일 연결

선택자[속성명="속성값"] { css 코드; } : 특정 속성을 선택

선택자1 > 선택자2 { css코드; } : 지정된 요소 바로 하위에 존재하는 요소를 선택

선택자1 선택자2 { css코드; } : 지정된 요소의 모든 하위에 존재하는 요소를 선택


:active : 클릭하고 있는 경우 반응
:hover : 마우스가 올라가는 경우 반응

:focus : 요소에 초점이 맞춰졌을 때 반응
:checked : 요소가 체크 되었을 때(radio, checkbox)
:enabled / :disabled : 요소가 사용 가능한/불가능한 상태일 때

A선택자 + B선택자 { CSS 코드; } : A 바로 뒤(다음)에 있는 B 요소 하나를 선택 (+)
A선택자 ~ B선택자 { CSS 코드; } : A 뒤에 있는 모든 B요소를 선택 (~ 틸드(Tilde))

선택자[속성명 ~= "특정값"] { CSS 코드; } : 띄어쓰기로 구분되어있는 여러 속성 값이 작성된 속성중
        속성값이 특정 값을 단어로 포함하는 요소를 선택
선택자[속성명 |= "특정값"] { CSS 코드; } : 속성값이 특정 값을 단어로 포함하는 요소를 선택
        단, "-" 기호로 구분, "-" 앞의 내용이 동일해야 한다.
선택자[속성명 ^= "특정값"] { CSS 코드; } : 속성값이 특정값으로 시작하는 요소를 선택(^ : 캐럿)
선택자[속성명 $= "특정값"] { CSS 코드; } : 속성값이 특정값으로 끝나는 요소를 선택
선택자[속성명 *= "특정값"] { CSS 코드; } : 속성값이 특정값을 포함하는 요소를 선택

p:first-child : 첫 번째 자식이 p가 맞으면 선택 
p:last-child : 마지막 자식이 p가 맞으면 선택
p:nth-child(수열) : 수열 번째 자식이 p가 맞으면 선택
p:nth-last-child(수열) : 마지막 자식부터 위로 수열 번째 자식이 p가 맞으면 선택

:first-of-type : 같이 선택된 형제들 중에서 첫 번째 요소
:last-of-type : 같이 선택된 형제들 중에서 마지막 요소
:nth-of-type(수열) : 같이 선택된 형제들 중에서 수열번째 모든 요소
:nth-last-of-type(수열) : 같이 선택된 형제들 중에서 뒤에서 수열번째 모든 요소

선택자1:not(선택자2) : 괄호 내 선택자를 제외한 나머지 요소를 선택
